# opencode-memoria

Repositório de memória persistente para o opencode.

## Como usar

Adicione no `opencode.json` de cada projeto:

```json
{
  "$schema": "https://opencode.ai/config.json",
  "instructions": [
    "https://raw.githubusercontent.com/Gabriel14785/opencode-memoria/main/preferencias.md",
    "https://raw.githubusercontent.com/Gabriel14785/opencode-memoria/main/projetos.md"
  ]
}
```

## Arquivos

- `preferencias.md` — Preferências pessoais, estilo de código, ferramentas favoritas
- `projetos.md` — Informações sobre projetos ativos e passados
- `decisoes.md` — Decisões técnicas e arquiteturais importantes
